BANTU — Banco da Nova Terra (Gerador Consagrado de Contas)
======================================================

Fase 1 (Manual Sagrado) -> Depois Automação Segura

Exemplo (Ministério):
python3 bantu_account_generator.py --id BANTU-0001 --nome "Igreja Trono e Glória" --tipo ministerio --address 0x0000000000000000000000000000000000000000 --out "/mnt/data/bantu_generator/output/contas/ministerios" --sig "/mnt/data/KIZEQUIEL ASSINATURA.jpg"

Pastas de saída:
- JSON   -> /mnt/data/bantu_generator/output/contas/ministerios/json
- PDF    -> /mnt/data/bantu_generator/output/contas/ministerios/pdf
- QRCode -> /mnt/data/bantu_generator/output/contas/ministerios/qrcodes
